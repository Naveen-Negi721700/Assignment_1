# Assignment 1 Website

A modern and responsive website built using **Next.js**, **React**, **Tailwind CSS**, and **MongoDB**.  
This project includes a fully responsive UI, contact form functionality, and database integration.

## 🚀 Live Demo

🔗 https://assignment-1-sooty-zeta.vercel.app/

## 📌 Features

- Responsive Navbar
- Modern UI Design
- Tailwind CSS Styling
- Contact Form
- MongoDB Database Integration
- Fully Responsive for Mobile, Tablet & Desktop
- Smooth User Experience
